## Developer documentation

### Quick Preview

```sh
docker run -p 8888:80 --rm ghcr.io/aecomet/gozoku
# => http://localhost:8888/
```

### Set up for local

```sh
# install node
node -v
# => 24.14.0

npm i -g pnpm

pnpm install

pnpm run dev
# => Local:   http://localhost:8888/
```
